Create an AI Experiment

Easily design and test AI-powered experiments. Follow the step-by-step process to define your experiment, customize its behavior, and share it with the community.

Step 1: Experiment Details

Provide key details about your experiment to ensure accurate analysis.

Next, provide a brief description of your experiment. You can enter it in either text or HTML formatâ€”simply use the toggle switch to select your preferred option.

Editor Mode Text

Step 2: User Experience

In this step, you will decide how users interact with your experiment. Think of it as designing how they will search, view, and engage with the results.

Simple Prompt UX is for engineering prompts that do not require data insertion from case data or other sources.

Preview Experience

OR use a custom UX, supporting up to 10 fields (TextField, TextArea, Json, DropDownList), each requiring an ID and Label. Currently defined in JSON, with a graphical editor coming soon. Custom UX is limited to Simple Experiments but will expand later.

Custom UX

Step 3: Prompt

The prompt defines the conversation flow by specifying system, user, and assistant messages in JSON format. First select a sample prompt you would like to base yours off of and then select which AI model to choose for your experiment.

You can then customize it for each stakeholder within that prompt. Drag and drop to reorder.

Custom UI Fields: How They Work

When you use the parameters below in your prompts, they will be automatically replaced with the matching user field values before being sent to AI for processing. If you plan to use reactive data, please check the additional requirement below.

Step 4: Parameter

This page allows you to customize the behavior of your AI experiment by adjusting key settings.

If you've used parameters, you can select a saved set from the dropdown.

Select Model of ChatGPT to use with your experiment

Creative level (Temperature Value):

Controls randomness. Higher values make AI responses more creative, while lower values make responses more predictable.

Creativity Level: 0.7/2

Response Diversity:

Limits word choices based on probability. A higher value makes responses more diverse, while a lower value makes them more focused.

Response Diversity: 0.95/1

Repetition Control (Frequency Penalty):

Reduces repeated words or phrases. Higher values discourage repetition.

Frequency Penalty: 0.95/1

Novelty Boost (Presence Penalty):

Encourages AI to use new words that haven't appeared before.

Novelty Boost: 0.51/1

Step 5: Contributors

Enter the email addresses of people you want to collaborate with.

How to add contributors:

  • Use this format: ["email1@example.com", "email2@example.com"]
  • Use real work emails, not personal or vanity addresses
  • Don't include yourself (since you own this experiment)

Important notes

  • Only the listed contributors can co-author your experiment.
  • Private experiments do not support contributors.
  • We're updating this soon to make it easier to search and add people.
